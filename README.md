# Student Performance Analysis Using Big Data

## Project Overview

The **Student Performance Analysis Using Big Data** project focuses on analyzing large student datasets to identify academic performance patterns and key factors influencing student success. The project applies data analysis techniques to extract meaningful insights from educational data.

This analysis helps understand how factors such as study hours, attendance, and demographics affect student performance.

## Objectives

* Analyze student academic records using data analytics techniques
* Identify factors that influence student performance
* Visualize patterns and trends in student scores
* Generate insights that can help improve academic outcomes

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
* Excel Dataset

## Dataset Description

The dataset contains student academic records including:

* Student ID
* Gender
* Study Hours
* Attendance
* Math Score
* Science Score
* English Score
* Average Score

The dataset is used to perform exploratory data analysis and identify performance trends.

## Key Analysis Performed

* Data cleaning and preprocessing
* Calculation of average student scores
* Distribution of student performance
* Analysis of study hours vs exam scores
* Gender-based academic performance comparison
* Identification of top-performing students

## Visualizations

The project includes several visualizations to understand data patterns:

* Score distribution histogram
* Study hours vs performance scatter plot
* Average score comparison by gender
* Subject-wise performance charts

These visualizations help interpret trends and relationships within the dataset.

## Insights

Some key insights discovered during the analysis:

* Students who study more hours generally achieve higher scores
* Consistent attendance positively impacts academic performance
* Some subjects show stronger score variations among students
